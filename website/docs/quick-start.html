Quick Start

Get your first scrape done in under a minute.

1

Navigate to any website

Open the page you want to scrape in your browser.

2

Open the extension

Click the extension icon to open the side panel. If a pre-built recipe is available for the site, it will be suggested automatically.

3

Pick elements to scrape

Use the visual element picker to click on the data you want. The extension will automatically generate CSS selectors and detect similar elements.

4

Preview and export

Review the extracted data in the preview table, then export it as CSV or Excel.

Pro Tip: Save your scrape configuration as a recipe to quickly re-run the same scrape anytime.
