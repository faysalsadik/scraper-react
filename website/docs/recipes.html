Recipes

Recipes are reusable scraping templates. Each recipe stores the target URL pattern, fields to extract, and pagination settings so you can re-run the same scrape later.

Creating a recipe

  1. Open the side panel on your target page
  2. Click New Recipe and give it a name
  3. Add fields using the element picker
  4. Optionally configure pagination
  5. Save the recipe — it's stored locally in your browser

Managing recipes

  • View all saved recipes from the popup or side panel
  • Edit field selectors, names, and types anytime
  • Delete recipes you no longer need
  • Recipes support up to 20 fields each
